Formulários

O atributo Action serve para definir o URL no qual os dados recolhidos do formulario serão enviado quando o user o enviar Fonte: developer.mozilla

O atributo Method serve para designar qual método Http os dados serão enviados, sendo eles:

  • GET. : serve como parâmentro de consultas, ou seja, os dados do formulário são visíveis na URL, geralmente usado quando para realizar solicitações de busca no servidor.
  • POST. : usado quando os dados fornecidos no formulário são incluídos no corpo da solicitação Http, sendo ele mais adequado para enviar dados/informações mais sensíveis como senhas ou grandes quantidades de dados, uma vez que quando o método Post é usado os dados não ficam visíveis na URL.
