Tipos de Campo Input

Exemplos

  1. Campo de input text é usado para receber informações do tipo texto.



  2. Campo de input email usado para coletar endereços de email dos usuários em um formulário, alguns navegadores formatam o email automaticamente



  3. Campo de input senha é usado para coletar a senha do usuário, onde o texto é inserido e automaticamente mascarado.



  4. Campo de input number é um tipo de campo numérico, onde o user pode inserir valores numéricos, podendo usar setas para cima e para baixo, para aumentar ou diminuir o valor inserido



  5. Campo de input date é um tipo de campo para entrada de data, permite os users selecionar uma data no calendário



  6. Campo de input checkbox serve para criar uma caixa de seleção que os users podem marcar ou desmarcar



  7. Campo de input radio é um tipo de campo que cria botões de opções, que permitem ao user escolher uma única opção



  8. Campo de input file é um tipo de campo que permite realizar upload de arquivos que o user selecionar



  9. Campo de input submit serve para criar um botão de envio que, ao clicado, envia os dados inseridos para o servidor



  10. Campo de input button serve para criar um botão interativo que pode ser usado com javaScript para executar ações


  11. Campo de input hidden serve para criar um campo oculto, que não é exibido para os users, mas sim para armazenar valores que serão enviados junto ao formulário


  12. Campo de input color serve para criar um campo de seleção de cor que permite aos users escolherem uma cor


  13. Campo de input range serve para criar um controle deslizante que permite ao user selecionar um valor dentro de um intervale pré definido


  14. Campo de input Search serve para criar um campo de pesquisa que pode ser estilizado de maneira específica e oferece recursos de pesquisa


  15. Campo de input image serve para o user inserir um imagem no forms, podendo ser especificado o tipo de imagem



Atributos dos campos de Imput
  1. Type: especifica o tipo de entrada
  2. Name: define o nome da entrada, que é usado para identificar os dados
  3. Id: fornece um identificador único para a entrada
  4. Value: define um valor padrão para entrada
  5. Placeholder: insere um texto temporário dentro do campo
  6. Required: indica que o campo é obrigatório
  7. Autofocus: define foco automático para o campo quando a pag é carregada
  8. Disabled: desativa o campo, tornando-o inacessível ao user
  9. Readonly: torna o campo somente leitura
  10. Autocomplete: controla a funcionalidade de preenchimento automatico para a entrada
  11. maxlength: limita o número máximo de caracteres que podem ser inseridos
  12. minlength: define o número mínimo de caracteres que podem ser inseridos
  13. pattern: especifica uma expressão regular que o valor inserido na entrada deva atender
  14. size: define o tamanho visual de cada entrada
  15. step: define o intervalo entre os valores válidos em campor númericos
  16. accept: especifica os tipos de arquivos aceitos para campos de upload de arquivos
  17. multiple: permite a seleção multipla de arquivos em campos de upload de arquivo
  18. min e max: definem os valories min e max permitidos para campos numéricos
  19. checked: marca um checkbox por padrão
  20. arial-label: fornece uma descrição acessível para a entrada para usuários de tecnologias assistivas
